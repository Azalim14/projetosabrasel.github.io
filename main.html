Documentos complementares

Esses documentos te auxiliarão nas etapas do projeto.

Fluxo de Gestão de Projetos Instalação de Barra de Ferramenta AbraMetas Capacitação Trello Boas práticas Elaboração Canvas

  • Primeira Etapa | Fase de pré-projeto

    Preencher Pré-projeto

    O Solicitante cria a ideia de projeto e preenche a cartilha de pré-projeto.

    Pré-projeto

  • Segunda Etapa | Fase de pré-projeto

    Categorizar pré-projeto

    O Escritório de Projetos avalia a complexidade do projeto e define as ferramentas de gestão que serão aplicadas por meio da ferramenta de categorização.

    Formulário de Categorização

  • Terceira Etapa | Fase de pré-projeto

    Apresentar pré-projeto

    O Solicitante irá apresentar a ideia de projeto para a Diretoria e Presidência.

  • Quarta Etapa | Fase de pré-projeto

    Aprovar projeto e nomear líder

    A Diretoria + Presidência deverá aprovar o projeto e nomear o líder. Caso o projeto seja aprovado, o Escritório de Projetos deverá incluir o projeto na Carteira de Projetos Ativos. Caso não seja aprovado, entrará para a lista de futuros projetos.

  • Quinta etapa | Fase de concepção e iniciação

    Conceber o projeto

    O líder do projeto desenvolve a ideia do projeto finalizando o preenchimento do Canvas.

    Modelo de Canvas

  • Sexta Etapa | Fase de concepção e iniciação

    Elaborar o Termo de Abertura

    O Líder do Projeto formaliza o início do projeto, define a sua meta e seus critérios de encerramento por meio do Termo de Abertura.

    Termo de Abertura

  • Sétima Etapa | Fase de concepção e iniciação

    Validar Canvas e Termo de Abertura

    O escritório de projetos juntamente ao líder valida o Canvas e o Termo de Abertura.

  • Oitava Etapa | Fase de concepção e iniciação

    Definir a equipe do projeto

    O Líder do Projeto define os membros que irão apoiar na execução das atividades do projeto e realiza as contratações necessárias.

  • Nona Etapa | Fase de concepção e iniciação

    Comunicar com os envolvidos

    O líder comunica o início do projeto para as partes envolvidas.

  • Décima Etapa | Fase de Planejamento

    Detalhar o escopo do projeto

    O Líder em conjunto com sua equipe detalham as principais entregas do projeto por meio da Estrutura Analítica do Projeto (EAP).

    Escopo do Projeto

  • Décima primeira Etapa | Fase de Planejamento

    Aprovar plano de projeto

    A Diretoria + Presidência aprova o plano do projeto.

  • Décima segunda Etapa | Fase de Planejamento

    Definir as atividades a serem realizadas

    O Líder em conjunto com sua equipe incluem as atividades a serem executadas no Trello (caso o projeto seja de baixa ou média complexidade) atribuindo prazo e um único responsável por ação. Nos projetos de alta complexidade, o cronograma das atividades será colocado no Project.

  • Décima terceira Etapa | Fase de Planejamento

    Realizar kickoff

    O Líder do Projeto apresenta - em uma reunião com os stakeholders - o plano definido para a execução do projeto.

    Modelo de Kickoff

  • Décima quarta Etapa | Fase de Execução e Controle

    Controlar a execução das atividades

    O Líder controla periodicamente o andamento das atividades do projeto, propõe contramedidas em casos de riscos e atrasos e divulga o Status Report.

    Modelo de Status Report alta complexidade Modelo e Status Report baixa/alta complexidade

  • Décima quinta Etapa | Fase de Encerramento

    Encerrar o projeto

    Após ter realizado 100% do escopo planejado, o Líder elabora o Termo de Encerramento para formalizar a conclusão do projeto perante a organização.

    Termo de encerramento
