Documentos complementares

Esses documentos te auxiliarão nas estapas do projeto.

Fluxo de Gestão de Projetos Capacitação Trello Instalação de Barra de Ferramenta AbraMetas

  • Primeira Etapa | Fase Iniciação

    Conceber o projeto

    Os diretores e PMO se reúnem para desenvolver e esboçar a ideia do projeto por meio do Canvas

    Canvas

  • Segunda Etapa | Fase Iniciação

    Priorizar o projeto

    O PMO em conjunto com os Diretores avaliam a prioridade do projeto para a organização, utilizando a ferramenta de priorização.

    Priorização

  • Terceira Etapa | Fase Iniciação

    Categorizar o projeto

    O PMO avalia a complexidade do projeto e define as ferramentas de gestão que serão aplicadas por meio da ferramenta de categorização.

    Categorização

  • Quinta Etapa | Fase Iniciação

    Nomear líder do projeto

    O Diretor nomeia o Líder do Projeto, isto é, o responsavel pelo atingimento da meta do projeto.

  • Sexta Etapa | Fase Iniciação

    Definir a equipe do projeto

    O Líder do Projeto define os membros que irão apoiar na execução das atividades do projeto

  • Sétima Etapa | Fase Iniciação

    Elaborar o Termo de Abertura

    O Líder do Projeto formaliza o início do projeto, define a sua meta e seus critérios de encerramento por meio do Termo de Abertura.

    Termo de Abertura

  • Oitava Etapa | Fase Planejamento

    Detalhar o escopo do projeto

    O Líder em conjunto com sua equipe detalha as principais entregas do projeto por meio da Estrutura Analítica do Projeto (EAP).

    Modelo

  • Nona Etapa | Fase Planejamento

    Definir as atividades a serem realizadas

    O Líder em conjunto com sua equipe incluem as atividaes a serem executadas no Trello atribuindo prazo e um único responsável por ação.

    Modelo

  • Décima Etapa | Fase Planejamento

    Realizar kick off

    O Líder do Projeto apresenta para todos os stakeholders o plano definido para a execução do projeto.

    Modelo de Kick off

  • Décima primeira Etapa | Fase Execução e Controle

    Controlar a execução das atividades

    O Líder controla periodicamente o andamento das atividades do projeto, propõe contramedidas em casos de atrasos e divulga o Status Report.

    Status Report de Baixa Complexidade Status Report de Alta Complexidade

  • Décima segunda Etapa | Fase Encerramento

    Encerrar o projeto

    Após ter realizado 100% do escopo planejado, o Líder elabora o Termo de Encerramento para formalizar a conclusão do projeto perante a organização.

    Termo de Encerramento
